# Ultimate Privacy App Example

Complete example showing how to use Ghost SDK with all privacy features.

## Features Demonstrated

- ✅ Ghost ZK proofs
- ✅ Monero ring signatures
- ✅ Zcash shielded assets
- ✅ Private transfers
- ✅ Token issuance
- ✅ React components

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Usage

```tsx
import { PrivateWallet } from '@ghost-sdk/react';

function App() {
  return <PrivateWallet />;
}
```

That's it! Full privacy in 3 lines.

## What's Included

- Private wallet UI
- Balance display (hidden)
- Transfer form
